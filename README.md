### Requirements
- OpenJDK 17
- Docker
- Code Editor

### Run development
- run `./bootstrap` first, to deploy local `postgres` and `localstack`
- `mvn spring-boot:run`
- It will run on port `8082`
- Check the [API docs](http://localhost:8082/swagger-ui/index.html)
